### 21_point
Классическая карточная игра “21 очко” с консольным управлением.  
---

### Wordle
Игра-угадайка слов (аналог популярной Wordle).  

---

### Мини-игры и утилиты
- Shifr  — шифр Цезаря для русского и английского текста  
- Case  — смещение букв в словах на длину слова  
- Magic 8 Ball  — магический шар, отвечающий на вопросы “Да/Нет”

---

### Snake
Классическая змейка с уровнями сложности, бонусами и рекордом.  

---

### Tello
Программа для управления дроном Tello через Python.
